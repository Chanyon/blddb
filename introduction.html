说明

一、角块公式库的生成主要分为筛选和排序两个步骤。

1、筛选规则：
  • stm ≤ 17, qtm ≤ 20.
  • 只含 R*、 U*、 D*、 F*、 B*, 不含 L*、内层、双层和转体等操作.
  • |R2| ≤ 6, |U2| ≤ 4, |D2| ≤ 2, |F2| ≤ 2, |B2| ≤ 2.
  • |D*| ≤ 6, |F*| ≤ 3, |B*| ≤ 3.
  • F* 和 B* 不同时出现.
  • 起手为上、中、下之一，起落手一致，中途不换手.
  • 在手位为上时，不做 U2 或者 F2.
  • 在手位为中时，不做 B* 或者 F*.
  • 在手位为下时，不做 U2 或者 B2.
  • 在手位为后时，不做 U2 或者 D2 或者 B* 或者 F*.
  • 至多一次成为后手位，且只能在后手位时完成一个步骤.
2、排序规则：
    每条公式都有一个从多方面进行评价的“等效步数”。角块公式库的排序主要以噩梦公式集为训练样本，通过模拟退火算法对权重值进行优化，使得总体而言噩梦公式在所有备选公式中的排序尽量靠前。排序前18条的公式会被显示在网页上。噩梦公式会被涂上灰色（#D0D0D0）。

二、翻角公式库的生成和角块公式库的生成基本一样。区别在于 stm ≤ 18 和只包含 R*、 U*、 D*.

三、棱块公式库和翻棱公式库仅包含一些旧公式和旧公式的衍生公式（例如镜像，U 和 U' 互换等）。这是因为在类似的体系和限制下，棱块每种状态的公式数比角块多很多，暂时无法做到像角块一样暴力搜索出全部解法。排序仅通过 stm 升序排列，排序前18条的公式会被显示在网页上。噩梦公式会被涂上灰色（#D0D0D0）。

四、编码的设置：本网页的默认编码是彳亍编码，已经预提供了彳亍编码和 Speffz 编码，可一键切换。其它编码体系的用户也可直接在网页上进行修改。
